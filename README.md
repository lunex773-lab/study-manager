# 受験進捗トラッカー（study-manager）

スマホ向けに作られた、大学受験のための**学習管理アプリ**です。試験までのカウントダウン、科目・分野ごとの習熟度、学習時間の記録、間隔をあけた復習（スペースドリピティション）までを 1 つにまとめています。

**依存ライブラリなし・ビルド不要**。`index.html` 1 ファイルだけで動き、データはすべてブラウザの `localStorage` に保存されます。サーバーもアカウント登録も不要です。

---

## 特長

- 📱 **単一ファイル / 依存ゼロ** — バニラ JavaScript のみ。`index.html` を開くだけで動作します
- 🔒 **完全ローカル保存** — データは端末のブラウザ内だけに保存。外部送信は一切ありません
- ⏱️ **カウントダウン / ストップウォッチ式タイマー** — 15・25・30・45・60・90 分のプリセットで学習時間を計測、あとから手動記録も可能
- 🎯 **試験日カウントダウン** — 共通テスト・二次・模試など複数登録でき、本番まで「あと◯日」を表示
- 📚 **科目 × 分野の習熟度管理** — 分野ごとに `未着手 / 学習中 / 要復習 / 完璧` の 4 段階で進捗を管理
- 🔁 **復習リマインド** — 記録した学習を「1 日後・1 週間後・1 ヶ月後」に自動で復習リストへ。溜まると通知バッジ表示
- 🔥 **連続学習ストリーク & ヒートマップ** — 直近 15 週の学習量をカレンダー状に可視化
- 📈 **統計** — 週 / 月 / 目標期間ごとの学習時間や科目別の内訳を集計
- 🏁 **期間目標** — 「夏期に◯時間」などの目標を設定して達成度を追跡
- 🎨 **8 種のテーマ** — midnight / daybreak / sakura / carbon / ocean / forest / grape / custom。アクセントカラーもカスタム可能
- 🔊 **効果音・触覚フィードバック** — 完了時のサウンドやバイブレーション（オン/オフ切替可）
- 💾 **JSON エクスポート / インポート** — 機種変更やバックアップに対応

## 使い方

サーバーは不要です。以下のいずれかの方法で開けます。

**ローカルで開く**

```bash
# リポジトリを取得して index.html をブラウザで開くだけ
git clone https://github.com/lunex773-lab/study-manager.git
cd study-manager
open index.html      # macOS（Linux は xdg-open、Windows は start）
```

**Web に公開する（GitHub Pages など）**

`index.html` を任意の静的ホスティングに置くだけで公開できます。GitHub Pages なら、リポジトリの Settings → Pages で `main` ブランチを指定すれば URL が発行されます。

**スマホで使う**

公開した URL をスマホのブラウザで開き、「ホーム画面に追加」するとアプリのように全画面で利用できます（`apple-mobile-web-app` 対応、セーフエリア対応済み）。

## 画面構成

下部のタブで 5 つの画面を切り替えます。

| タブ | 内容 |
| --- | --- |
| **ホーム** | 試験カウントダウン、ストリーク、学習ヒートマップ、復習の通知 |
| **科目** | 科目と分野の追加・並べ替え、分野ごとの習熟度チェック |
| **復習** | 期日になった復習の一覧と完了操作（バッジで件数表示） |
| **記録** | タイマーで学習を計測、または手動で記録を追加 |
| **統計** | 期間別の学習時間・科目別内訳のグラフ |

右上の歯車から、志望校・試験日の管理、テーマ、効果音、目標、データのエクスポート/インポートなどの設定にアクセスできます。

## データについて

- 保存先はブラウザの `localStorage`（キー: `study-tracker-v1`）です
- **端末・ブラウザをまたいでの同期はありません。** 機種変更の際は設定画面から **エクスポート → インポート** でデータを移行してください
- プライベートブラウズ（シークレットモード）ではデータが保存されない場合があります
- 旧バージョン（`kobe-study-tracker:v1`）のデータは初回起動時に自動で引き継がれます

## 技術構成

- **言語 / スタック**: HTML + CSS + バニラ JavaScript（フレームワーク・ビルドツールなし）
- **永続化**: `localStorage`（スキーマ v2、JSON）
- **画面遷移**: 独自のクライアントサイド・ルーティング + トランジション
- **外部依存**: なし（アイコンはインライン SVG、フォントはシステムフォント）
